# AI-Powered-Data-Cleaning-Automation
“AI-powered automated data cleaning and EDA pipeline using Python. Detects dataset errors, fixes inconsistencies, and generates insights with Gemini.”
This project builds an AI-powered automated data-cleaning and EDA pipeline. 
It uses Python and Google Gemini to automatically detect data errors, fix inconsistencies, handle missing values, clean messy datasets, generate visualizations, and produce complete EDA insights & business recommendations. 
Designed for end-to-end automation — upload dataset → Gemini cleans it → EDA runs → insights generated instantly.
